项目名称:重庆单招高校预计录取查询系统<br>
==========

本系统目前包含了计算机类、园林类的历史录取最低分数据

相关分数数据源于"青云汇"

相关院校数据源于重庆市教育委员会

本项目仅为重庆市应届单招生提供录取信息填报参考
